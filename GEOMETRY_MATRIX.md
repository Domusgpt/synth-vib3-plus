# Synth-VIB3+ Geometry Matrix

## The 72 Unique Sound+Visual Combinations

This document details all 72 combinations in the Synth-VIB3+ three-dimensional matrix system.

## Matrix Structure

```
3 Visual Systems × 3 Polytope Cores × 8 Base Geometries = 72 Combinations
```

### Dimension 1: Visual System (Sound Family)
- **Quantum**: Pure harmonic synthesis (sine-dominant, high Q: 8-12)
- **Faceted**: Geometric hybrid synthesis (square/triangle, moderate Q: 4-8)
- **Holographic**: Spectral rich synthesis (sawtooth, low Q: 2-4, high reverb)

### Dimension 2: Polytope Core (Synthesis Branch)
- **Base** (Geo 0-7): Direct synthesis with filtering
- **Hypersphere** (Geo 8-15): FM synthesis
- **Hypertetrahedron** (Geo 16-23): Ring modulation

### Dimension 3: Base Geometry (Voice Character)
- **Tetrahedron** (0): Fundamental, minimal filtering
- **Hypercube** (1): Complex, dual oscillators with detune
- **Sphere** (2): Smooth, filtered harmonics
- **Torus** (3): Cyclic, rhythmic phase modulation
- **Klein Bottle** (4): Twisted, asymmetric stereo
- **Fractal** (5): Recursive, self-modulating
- **Wave** (6): Flowing, sweeping filters
- **Crystal** (7): Crystalline, sharp attack transients

## Geometry Index Calculation

To calculate the full geometry index from visual system and base geometry:

```dart
// Visual System Offsets
int getSystemOffset(String system) {
  switch (system) {
    case 'quantum':      return 0;  // Geometries 0-7
    case 'faceted':      return 8;  // Geometries 8-15
    case 'holographic':  return 16; // Geometries 16-23
  }
}

// Full Geometry Index
int fullGeometry = systemOffset + baseGeometry; // 0-23

// Decompose Geometry Index
int coreIndex = fullGeometry ~/ 8;      // 0=Base, 1=Hypersphere, 2=Hypertetrahedron
int baseGeometry = fullGeometry % 8;    // 0-7 (Tetrahedron through Crystal)
```
